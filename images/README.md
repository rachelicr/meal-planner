# Meal Images

Drop your meal photos in here.

- Any format works: .jpg, .jpeg, .png, .webp
- Keep file sizes reasonable — 300-500kb is plenty for a phone screen
- Name them clearly: yoghurt-jar.jpg, salmon-lentils.jpg etc

Then add the filename to the `image` column in recipes.csv:

    breakfast,b1,Yoghurt Jar,false,"Full fat yoghurt...","yoghurt|blueberries",images/yoghurt-jar.jpg

Leave the image column blank for meals without a photo — the app handles it gracefully.
